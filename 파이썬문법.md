# 파이썬 문법

## 문자열

문자열은 수정이 불가능한 immutable이다.

## 이스케이프 코드 정리

\n 줄바꿈

\t 탭

\\ 슬래시

\' 작은따옴표

\" 큰따옴표

\r 커서를 가장 앞으로 이동

\f 현재 커서를 다음 줄로 이동

\a 벨 소리

\b 지움

\000 널

## 문자열 포맷팅

%s 문자열

%c 문자 1개

%d 정수

%f 실수

%o 8진수

%x 16진수

%% '%'

%10s 길이 10인 문자열

%.4f 소수 점 4자리 실수

## format 함수 포매팅

{0:<10} 왼쪽 정렬 길이 10

{0:>10} 오른쪽 정렬 길이 10

{0:^10} 가운데 정렬 길이 10

{0:=^10} 빈 자리는 '='으로 채워넣음 가운데 정렬 길이 10

{0:10.4f} 소수점 4자리실수 길이10

{{}} {}

## f 포맷팅

f'나는 내년이면 {age+1}살이 된다'

f'{"hi":!^10}' hi 문자열 빈자리 '!' 가운데정렬 길이 10

f'{3.42134234:10.4f}'

f'{{ and }}'

## 문자열 관련 함수

.count('문자') 찾을 문자 개수 반환

.find('문자') 문자 처음 나온 인덱스 반환 없으면 -1 반환

.index('문자') 문자 처음 나온 인덱스 반환 없으면 에러

'삽입할 문자'.join(문자열, 리스트, 튜플) 삽입할 문자가 사이사이에 껴있는 문자열 반환

.upper() 대문자만들기

.lower() 소문자만들기

.lstrip() 왼쪽 공백 삭제

.rstrip() 오른쪽 공백 삭제

.strip() 공백 삭제

.replace("문자열 일부분", "바꿀 문자열") 문자열 일부분을 바꿀 문자열로 바꿔준다

.split("문자") 문자를 기준으로 쪼개어 리스트로 반환

## 리스트

리스트에는 어떤 자료형도 요소로 넣을 수 있다.

리스트는 값을 수정하거나 삭제할 수 있다.

## 리스트 문법

del 객체 객체를 삭제한다

.append(추가할 자료) 리스트 맨 마지막에 추가한다

.sort() 리스트를 정렬한다.

.reverse() 리스트를 역순으로 뒤집는다.

.index(자료) 자료의 인덱스를 반환한다. 없으면 에러

.insert(인덱스 위치, 자료) 인덱스 위치에 자료를 삽입한다.

.remove(자료) 첫번째로 나오는 자료를 삭제한다.

.pop() 맨 마지막 요소를 반환하고, 리스트에서 삭제한다

.pop(인덱스) 인덱스에 위치한 걸 끄집어 내고, 리스트에서 삭제한다.

.count(자료) 자료가 몇개있는지 세어준다.

.extend(리스트) 리스트에 리스트를 더한다

## 튜플

튜플은 값을 변경할 수 없는 immutable 이다.

한개만 있을 경우 무조건 , 를 붙여야하고 ()를 생략할 수 있다.

## 딕셔너리

key와 value로 구성되어있고, key를 통해 value를 얻는다.

a[키] = 밸류 로 a 딕셔너리에 추가할 수 있다.

문자열, 리스트, 튜플 처럼 인덱싱을 사용할 수 없다. 순서가 없음

key는 리스트가 될 수 없다. 왜냐하면 키는 변하면 안되기 때문이다.

## 딕셔너리 문법

a = {'name' : 'pey', 'phone' : '0119993323', 'birth' : '1118'}

a.keys() 딕셔너리 키 객체를 반환한다.

a.value() 딕셔너리 밸류 객체를 반환한다

a.items() 딕셔너리 key, value 튜플로 묶은 객체를 반환한다.

.pop(키) 키 값을 끄집어내고, 딕셔너리에서 삭제한다.

.clear() 딕셔너리 지우기

.get(키) 밸류 반환 키가 없을 경우 None을 반환

.get(키, 디폴트 값) 밸류 반환 키가 없을 경우 디폴트 값 반환

키 in 딕셔너리 딕셔너리 안에 키가 있는지 조사한다. 있으면 True 없으면 False 반환

.update(여러개의 자료) 여러개의 자료를 추가

.remove(자료) 자료를 제거

## 불

True, False

bool(자료) 자료가 참이면 true, 자료가 거짓이면 false를 반환한다.

## 변수

변수는 객체를 가리킨다.

id(변수) 변수의 메모리 주소 반환

a is b a와 b가 동일한 객체를 가리키고 있는가?

### 리스트를 서로 다른 메모리를 사용하면서 복사하려면 복사하지 않고, 똑같은 리스트를 새로 만들던가, 슬라이싱을 하던가 아니면 copy모듈의 copy 함수를 사용해야한다.

a = [1,2,3]

b = [1,2,3]

b = [:]

```
from copy import copy
a = [1,2,3]
b = copy(a)
```

a, b = b, a

서로 값을 변경
